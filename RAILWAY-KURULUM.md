# Railway.app ile Bot Kurulumu (En Kolay!)

Railway = Ücretsiz + 24/7 + Çok kolay! 🚀

---

## ⚡ 5 Dakikada Kurulum

### Adım 1: Railway'e Git
https://railway.app aç

### Adım 2: GitHub ile Giriş Yap
- "Continue with GitHub" tıkla
- Yetkileri onayla

### Adım 3: Yeni Proje Oluştur
1. "Create New Project" tıkla
2. "Deploy from GitHub repo" seç
3. Repository'ni seç (Blox Fruits Bot)

### Adım 4: Environment Variables Ekle
1. "Variables" sekmesine git
2. "Raw Editor" tıkla
3. Yapıştır:
```
DISCORD_TOKEN=your_bot_token_here
```
4. Token'ın yerine **gerçek token'ını** yapıştır

### Adım 5: Deploy Et
- Otomatik deploy başlayacak
- ~2 dakika sonra çalışır!

## 🆘 Sorun Giderme

### Bot başlamıyor
- Logs'a bak (kırmızı hata mesajı?)
- Token doğru mu? Kontrol et
- package.json var mı? Evet

### Token hatası
- Token'ı kopya-yapıştır (tırnak yok)
- Deploy et
- Bekle (~2 dakika)
